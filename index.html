Git

- git init

Iniciar un proyecto con git (agrega carpeta .git)


- git status

Para ver los archivos que hace falta añadir


- git add "nombre"

Añadir los archivos (de uno por uno)


- git add .

Añadir todos los archivos


- git commit

Subi los archivos


tecla i para insertar, tecla esc y luego :wq (write quit)

Ponerle el nombre al commit


- git commit -m "mensaje"

Poner el nombre o mensaje sin entrar al editor


- git config --global user.email "email@gmail.com"

Configurar el email


- git config --global user.name "nombre"

Configurar el nombre, si se omite --global, la configuración se pone solo para el repositorio


- git log

Historial de commits


- git diff index.html

Diferencia entre archivos


- git branch

Ver las ramas


- git branch "nombre"

Crear nueva rama


- git checkout -- "nombre del archivo"

Deshace los cambios de un archivo


- git checkout "nombre"

Entrar a una rama



- git remote add origin "link repositorio"

Link del repositorio


- git push -u origin master

Subir archivos al repositorio


- git push

Subir los archivos (yo lo he usado después de crear el repositorio)


- git pull

Descargar los cambios del github


- git clone link-repositorio

Descargar los archivos desde github
